# 🎉 AI Expense Tracker - v1.0.0 (First Release)

## 📱 Android App - First Release

**Release Date:** April 21, 2026  
**Version:** 1.0.0  
**Build Type:** Debug APK

---

## 📥 Download

**APK File:** `AI-Expense-Tracker-v1.0.0.apk` (3.9 MB)

### Installation Instructions:

1. Download the APK file from this release
2. On your Android device, go to **Settings → Security**
3. Enable **"Install from Unknown Sources"** or **"Allow from this source"**
4. Open the downloaded APK file
5. Tap **"Install"**
6. Open the app and start tracking your expenses!

## ✨ Features

### 🔐 Authentication
- User registration with email and password
- Secure login with JWT tokens
- Profile management

### 💰 Expense Management
- Add expenses with amount, category, and description
- View all expenses in a beautiful list
- Edit and delete expenses
- Real-time expense tracking

### 🤖 AI-Powered Insights
- AI analyzes your expenses
- Identifies productive vs unproductive spending
- Provides personalized suggestions

### 📈 Dashboard

- Budget utilization charts

## 🔧 Technical Details

### Frontend
- **Framework:** React 19 + Vite
- **UI Library:** Tailwind CSS
- **Icons:** Lucide React
- **Animations:** Framer Motion
- **HTTP Client:** Axios
- **Mobile:** Capacitor 6

### Backend
- **Runtime:** Node.js + Express
- **Database:** MongoDB
- **Authentication:** JWT + bcrypt
- **AI Integration:** Groq API
- **Hosting:** Render

### Mobile
- **Platform:** Android (Capacitor)
- **Min SDK:** 22 (Android 5.1+)
- **Target SDK:** 34 (Android 14)

## 📝 Known Limitations

- This is a **debug build** (not optimized for production)
- Backend may sleep after 15 minutes of inactivity (Render free tier)
- First API call may take 30-60 seconds to wake up the server
- No Google Play Store distribution yet
